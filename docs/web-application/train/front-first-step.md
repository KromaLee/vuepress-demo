## 2. 开发环境搭建

```
- 安装jdk1.8
- 安装svn小乌龟
- 安装idea（vscode、eclipse均可，本培训以idea进行）
- 下载Lombok插件
- 使用svn小乌龟下载项目
- idea导入项目（等待maven自动下载依赖）
- 配置idea自动更新
```
