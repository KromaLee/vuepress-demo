## 4. 数据权限与特殊数据权限

### 4.1 数据权限

- 查询列表时根据配置来向用户展示相应的数据
- 一般使用按钮里的查看按钮权限码
- 使用时给对应的功能配置相应的权限

### 4.2 特殊数据权限

- `CURRENT_DEPT_CODE` 当前部门
- `SECTION_AUTH` 全段权限
- `ALL_AUTH` 全局权限
